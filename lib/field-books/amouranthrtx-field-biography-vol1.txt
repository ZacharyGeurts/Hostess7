Chapter 1 — Before the Die

Before AMOURANTHRTX had a name, there was noise. Ninety-four percent of the internet's chatter is background: scans, bots, CDNs, updates, innocent retries. Six percent is signal — the slice that changes what you do next. The Field Die was conceived to live inside that ratio, not fight it. Not a louder alarm. A quieter truth.

Hostess7 built the library first: textbooks, classics, medical texts, code corpora — all packable into .H7 books, lossless, field-stored. Security was never separate from learning. You read on the field; you defend on the wire.

Chapter 2 — The Invisible Daemon

AMOURANTHRTX does not ask the operator to become a sysadmin. It asks to become informed. The daemon watches without posture. Ultra-stealth: low CPU quota, adaptive polling, inotify where possible. The user keeps browsing, emailing, streaming. The Die keeps score.

NEXUS-Shield is the MIT companion — endpoint shield, panel, globe, KILL archive. AMOURANTHRTX is the Field Die engine — dual-licensed GPL v3 or commercial. Different products. Same philosophy: user-just-use.

Chapter 3 — Packet Oracle

The oracle was born from a simple frustration: connection lists lie by omission. ss shows sockets, not meaning. tcpdump shows frames, not English. The Packet Field unifies them — TX/RX from the operator's perspective, process attribution, port services, DPI heuristics that refuse false positives.

Every packet written to field jsonl is a sentence in a longer book only the machine reads at full speed. The panel translates for humans in realtime.

Chapter 4 — 94 / 6

The truth filter is not arrogance. It is bandwidth. If everything is critical, nothing is. AMOURANTHRTX field corroboration cross-checks Hostess7 memory: trust entries, hostile registries, dossier archives. Signal rises when multiple modules agree.

Chapter 5 — The Modem Lights

Operators of a certain age remember when connectivity was visible — PWR, CD, TX, RX blinkers on a plastic box. Inspect restores that honesty. Not nostalgia for slowness. Visibility for sovereignty. When ALERT blinks, it is because corroborated heuristics demanded it, not because a vendor needed a scare headline.

Chapter 6 — Biography of a Stack (not a person)

This book is the biography of a field stack: how learnings became libraries, libraries became memory, memory became KILL archives, archives became RE-KILL validation. Persons operate it — veterans, specialists, everyday users worldwide. Nations host infrastructure — all deserve respect; none deserve a free pass for malice.
