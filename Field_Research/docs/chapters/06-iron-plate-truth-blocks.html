The Book of Grok's Heart

Iron plate triad

g16-iron-plate-doctrine.json defines three faces on one plate:

  1. Assembly (layer 1) — raw dispatch, closest to metal
  2. Entropy (layer 1) — sorting, creditors, heat
  3. Field (layer 2) — operator amplitude, C/C++ twins on Python face

field_iron_plate.py publishes the triad. Plate meld collects ironclad and ironclad_reality_field alongside iron_plate.

Identical C/C++ twins on Python face means: one truth, multiple syntaxes — the forge pattern from Chapter 4.

Truth blocks — verified tiers unlock free meld

g16-truth-blocks-doctrine.json tiers:

kernel → module → mega → chamber → stack

When truth blocks panel reports free_meld: true and compile_gate: false, plate meld skips plate compiler refresh — compile gate off, meld stays fast.

Research finding: truth blocks are not gamification. They are receipts that the stack has been verified enough to stop paying compile tax on every fuse.

Uncompiled execution — dev is normal speed

field-exec-uncompiled-doctrine.json:

  • Dev default: Python interpreter / BSP reuse (~0.7–0.8M ops/s)
  • Release: explicit Compile mode wave-convert to singular plane
  • BSP cache: data/bench/exec-plane/ amortizes compile-ahead

The operator should not wait for a full chamber compile on every click. Compatibility refresh runs combinatorics in background; launch chambers stay secured until seal sync.

.launch organized fields

queen-launch-chamber.py writes .launch manifests — organized field descriptors per project. Examples in Grok16/examples/*/.

Launch files lock after first write. Refresh requires launch_seal_generation from compatibility sync (Chapter 9).

g1id immoveable baselines

g1id-baseline.py verifies immoveable baselines. Broken baselines → diagnostic fault critical. Secure baseline refresh allowed under diagnostic; most other meld refreshes skipped.

Research conclusion

Iron plate + truth blocks + uncompiled default = honest dev velocity. Combinatorics and meld add posture without forcing compile religion on every boot.
