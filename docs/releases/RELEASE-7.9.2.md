# NEXUS-Shield 7.9.2 — Tray tab picker · 0.25m triangulation

## System tray

- **`lib/panel-tray.py`** — Ayatana taskbar icon near the clock
- **Right-click** → pick any panel tab (Command, Library, Signals, Threats map, …)
- **`./nexus.sh`** starts tray after opening the browser
- **`./nexus.sh --tab library`** — open a tab directly from CLI

## GPS

- Triangulation CEP **0.25m** (`NEXUS_FIELD_TRI_CEP_M=0.25`)
