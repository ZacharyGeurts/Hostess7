## What this is

## Quick start (Linux x86_64)

```bash
# From release assets — H7e archive
./scripts/field-h7e-extract.sh hostess7-1.0.0-beta-source.h7e

cd hostess7-1.0.0-beta
./scripts/wire-stack.sh
sudo ./install-all.sh
./Hostess7/Hostess7.sh on
```
