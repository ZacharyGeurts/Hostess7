Coming from another browser?

Queen is the only browser product in AmmoOS. Your habits still work — tabs, bookmarks, passwords — but navigation runs through field gates on loopback.

  • Import — click IMPORT in the browser bar, or drop bookmarks.html / passwords.csv into .nexus-state/imports/
  • Default browser — Shift+click IMPORT to register Queen as your system default
  • Legacy profiles — Queen can sweep old gecko-family profiles locally; nothing is sent to the cloud

How the stack is layered

Everything is built bottom-up. Hardware is never broken — we witness and wire, we do not brick your machine.

  1. NEXUS C2 (:9477) — command, gates, trust strike, all security fixes
  2. ZNetwork — 100% internet pipe, relayer + exploit shield help Queen defend
  3. Queen CANVAS renderer — AMOURANTHRTX display technology (CANVAS.comp); chip layer, not a GUI
  4. Queen Browser (:9481) — secure rewritten shell; tabs, import, gates
  5. AmmoOS — inside Queen (Start tab) — not a separate browser product

Security you can see
