Kill Grok Orphans

Always-on watchdog that terminates reparented Grok orphan processes before they pile up at 30% CPU each.

GitHub Releases Wiki Grok16

Problem

Scripts like grok-firmware-audit.sh and Grok16/bin/awk bash wrappers can leak from boot-simulate.sh, reparent to init, and multiply — hundreds of orphans, each burning CPU.

Kill Grok Orphans scans every 5 seconds, matches known signatures, and kills them with SIGTERM → SIGKILL.

Install

Linux (native kgo, Grok16-built)

curl -fsSL https://github.com/ZacharyGeurts/Kill-Grok-Orphans/releases/latest/download/kgo-1.0.0-linux-x86_64.tar.gz | tar -xz
sudo packaging/linux/install.sh

macOS

git clone https://github.com/ZacharyGeurts/Kill-Grok-Orphans.git
cd Kill-Grok-Orphans && sudo packaging/macos/install.sh

Windows (Administrator PowerShell)

git clone https://github.com/ZacharyGeurts/Kill-Grok-Orphans.git
cd Kill-Grok-Orphans
.\packaging\windows\Install-KGO.ps1

Authentication

Killing reparented orphans requires elevated privileges on every OS:

OSAuthDaemon
Linuxsudosystemd (root)
macOSsudolaunchd (root)
WindowsAdministratorScheduled Task (SYSTEM)

Patterns killed

  • grok-firmware-audit.sh
  • Grok16/bin/awk
  • Grok16/bin/ shell shims
  • boot-simulate.sh orphans

Build with Grok16

export GROK16_ROOT=/path/to/Grok16
make && sudo make install

Compiler: g16 -std=gnu17 -O2 (Grok16 16.2.0)
